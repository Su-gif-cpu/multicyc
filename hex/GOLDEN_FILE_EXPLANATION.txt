关于 .golden 文件的说明
==========================

.golden 文件的作用：
- **黄金参考**（Ground Truth）：存储执行完成后的期望结果
- **自动化测试验证**：自动比对仿真结果与golden，判断通过/失败
- **格式**：通常是 "寄存器号 期望值"，如：
  ```
  3 0000dead
  5 000000ab
  ```

你的理解是对的：
✓ 对于手动查看和调试，只需要.hex（机器码）
✓ 在指令集模拟器中可以手动检查各寄存器最终值
✓ 如果要做自动回归测试才需要.golden

2. 为所有16个测试生成 .s 汇编源文件（用#注释）
3. 取消生成.golden文件（因为不需要）

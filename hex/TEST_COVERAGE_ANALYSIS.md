**缺陷评估**:
- ⚠️ sra测试结果中x12=0x00000008, x15=0xfffffff0表示有正负数右移测试，但覆盖不完整

## 缺陷汇总与建议

### 严重缺陷 (必须补充)

#### 1. ⚠️ 移位量边界值测试 (t01中需增强)
**缺陷**: sll, srl, sra都未测试移位量=0和移位量=31的极端情况
- 移位量=0: 应返回原值不变
- 移位量=31: 32位字的最大移位

**建议新增测试**:
```
sll x1, x1, 0    # 移位量=0，x1不变
sll x1, x1, 31   # 移位量=31
sra x1, x1, 0    # sra移位量=0
sra x1, x1, 31   # sra移位量=31（全为符号位）
```

#### 2. ⚠️ sra负数右移完整性测试 (t01)
**缺陷**: t01的sra测试结果不够完整
- x15=0xfffffff0 表示有负数sra测试，但补位(补1)完整性需验证

**建议**:
```
# 负数算术右移(应补1)
sra x1, -1, 1     # 0xFFFFFFFF >> 1 = 0xFFFFFFFF (补1)
sra x1, -2, 1     # 0xFFFFFFFE >> 1 = 0xFFFFFFFF (补1)

# 正数算术右移(应补0)
sra x1, 0x80000000, 1  # 0x80000000 >> 1 = 0xC0000000 (补1)
```

#### 3. ⚠️ 算术指令边界值 (缺少t02补充)
**缺陷**: 缺少负数+负数的结果测试

**建议新增测试**:
```
addi x1, x0, -1    # x1 = 0xFFFFFFFF
addi x2, x0, -2    # x2 = 0xFFFFFFFE
add x3, x1, x2     # x3 = 0xFFFFFFFD (负+负=负)

addi x4, x0, -1
addi x5, x0, 1
add x6, x4, x5     # x6 = 0 (负+正=0)
```

#### 4. ❌ t03中缺少负偏移的lw测试
**缺陷**: t03只测了正偏移，t12测了负偏移的sw/lw，但独立的lw负偏移完整性不足

**建议**: 虽然t12已有，但应补充独立的lw负偏移测试用例

#### 5. ⚠️ BNE的反向测试缺失
**缺陷**: t06有bne跳转满足的情况，但缺bne不跳转(相等)的测试

**建议新增**:
```
bne x1, x1, 8    # x1 == x1，不应跳转
```

## ⚠️ 严重缺陷 (必须补充)

### 1. 移位量边界值
**现状**: t01中移位指令使用动态移位量(x8)，但未测试特殊值
**缺失**:
- `sll x1, x1, 0` → 应返回原值
- `sll x1, x1, 31` → 最大移位
- `sra x1, x1, 31` → 全补位

**影响**: 移位指令边界缺陷是常见CPU设计问题

### 2. 算术指令负数组合
**现状**: t02只测了addi x1,x0,-1和addi x2,x1,1(得0)
**缺失**:
- 负+负 = 负
- 负-正 = 负
- 正-负 = 正

**影响**: 溢出检测和符号位传递的可靠性

### 3. BNE条件不满足
**现状**: t06_bne.hex中bne条件满足时跳转，但无反向测试
**缺失**: `bne x1, x1, offset` (相等时不跳转)

**影响**: 分支反向条件逻辑验证不完整

### 4. SRA补位完整性
**现状**: t01中x15=0xfffffff0可能有sra测试，但记录不清
**缺失**: 
- 负数sra右移必须补1: 0xFFFFFFFF >> 1 = 0xFFFFFFFF
- 正数sra右移必须补0: 0x7FFFFFFF >> 1 = 0x3FFFFFFF

**影响**: 算术右移的符号扩展功能
